# Raw Sources

This directory is the immutable evidence layer for the project.

Use it for:

- official PDFs
- guideline downloads
- textbook extraction notes
- clipped web articles
- source-specific working notes

Suggested subdirectories:

- `inbox/`: new material waiting to be processed
- `official/`: official PDFs and documents
- `extracted/`: extracted text from PDFs or portals
- `notes/`: source-by-source processing notes

Rules:

- do not rewrite the source itself
- do not treat this layer as publish-ready interpretation
- if a source is important enough to influence the site, summarize it in `knowledge-vault/` and normalize it into `src/content/`
