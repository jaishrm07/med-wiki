## Big picture

The cardiac cycle is not a list of events. It is a repeating pressure story.

## Clean way to think about it

Ask only three questions at each step:

- which chamber has higher pressure?
- which valves are open?
- what is blood doing right now?

## Common mistakes

- memorizing phases without pressure logic
- separating heart sounds from valve events
- forgetting atrial contraction is a late-diastolic event
