# Knowledge Vault

This is the maintained markdown wiki layer for the project.

It sits between `raw/` and `src/content/`.

Use it for:

- subject hubs
- system maps
- topic synthesis
- diagram notes
- presentation notes
- condition notes
- source summaries
- contradictions and open questions
- editorial drafts

The idea is simple:

- `raw/` stores evidence
- `knowledge-vault/` stores maintained understanding
- `src/content/` stores reviewed public content

This layer is intentionally Obsidian-compatible so the graph view and backlinks stay useful.
